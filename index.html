Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The img element will require a src attribute to determine the source of the file.
  • The section element often will have a class and id attribute to better connect with CSS and JavaScript.

CSS

  • The CSS is used to style the HTML elements using class and id attributes.

Git

  • Git uses the command line to save repo changes with version control.
  • Git Status, Git Checkout, Git Commit are three common commands made during the git workflow.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript is a popular programming language used in computer web apps.
  • JavaScript provides elements the ability for motion, query loops, and api functions.
